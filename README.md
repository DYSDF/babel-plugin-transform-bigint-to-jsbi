# Babel Plugin Transform BigInt To JSBI

> BigInt Syntax Polyfill

## Install

```bash
npm install babel-plugin-transform-bigint-to-jsbi
```

## Usage

```js
module.exports = {
  //...
  plugins: [
    //...
    'babel-plugin-transform-bigint-to-jsbi'
  ]
}
```

Conversion table please query [jsbi](https://www.npmjs.com/package/jsbi)
